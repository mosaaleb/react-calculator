## React Calculator
> A React powered calculator application.

#### Live Version
[Live](https://react-ulator.herokuapp.com/)

#### Installation
```
git clone git@github.com:mosaaleb/react-calculator.git
cd react-calculator
yarn install

# start development server
yarn react-scripts start
```
